# Sporting Goods Inventory Management System

A menu-driven command-line application that simulates a sporting goods store.
Users can browse product categories, view item details, purchase producs, and
track inventory updates in real time.

This project was implemented in **both Python and C++** to reinforce object-oriented
design principles across multiple programming languages.

---

## Features
- Object-oriented product hierarchy using inheritance and polymorphism
- Indoor and outdoor product categories with multiple item types
- Real-time inventory updates with stock validation
- Menu-driven CLI navigation
- Modular design with separated concern
